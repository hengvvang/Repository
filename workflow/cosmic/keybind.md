
### 基本系统功能

| 快捷键组合                   | 功能描述                       |
| ----------------------- | -------------------------- |
| `Super + Escape`        | 终止当前应用程序（Terminate）        |
| `Super + Alt + Escape`  | 注销用户（LogOut）               |
| `Super + Ctrl + Escape` | 进入调试模式（Debug）              |
| `Super` + `q`           | 关闭当前窗口（Close）              |
| `Super + `              | 锁屏（LockScreen）             |
| `Super + `1-9\`         | 切换到工作区 1-9                 |
| `Super + `0\`           | 切换到最后一个工作区（LastWorkspace）  |
| `Super + Print`         | 截图（Screenshot）             |
| `Super + t`             | 启动终端（Terminal）             |
| `Super + b`             | 启动网页浏览器（WebBrowser）        |
| `Super + f`             | 打开文件管理器（HomeFolder）        |
| `Super + space`         | 切换输入法（InputSourceSwitch）   |
| `Super + a`             | 打开应用程序库（AppLibrary）        |
| `Super + w`             | 显示工作区概览（WorkspaceOverview） |
| `Super + /`             | 打开应用启动器（Launcher）          |

### 音量和亮度控制

| 快捷键组合                   | 功能描述  |
| ----------------------- | ----- |
| `XF86AudioRaiseVolume`  | 音量增加  |
| `XF86AudioLowerVolume`  | 音量减少  |
| `XF86AudioMute`         | 静音    |
| `XF86AudioMicMute`      | 麦克风静音 |
| `XF86MonBrightnessUp`   | 亮度增加  |
| `XF86MonBrightnessDown` | 亮度减少  |
| `XF86AudioPlay`         | 播放/暂停 |
| `XF86AudioPrev`         | 上一首   |
| `XF86AudioNext`         | 下一首   |
| `XF86PowerOff`          | 关机    |

## 🖥️ 窗口管理类快捷键

### 聚焦操作

| 快捷键组合                        | 功能描述            |
| ---------------------------- | --------------- |
| `Super + Left/Right/Up/Down` | 焦点移动到左/右/上/下窗口  |
| `Super + h/j/k/l`            | 焦点移动（类似 Vim 风格） |
| `Super + u/i`                | 切换焦点到当前窗口的外层/内层 |


### 窗口移动

| 快捷键组合                                | 功能描述              |
| ------------------------------------ | ----------------- |
| `Super + Shift + Left/Right/Up/Down` | 移动当前窗口到左/右/上/下    |
| `Super + Shift + h/j/k/l`            | 移动当前窗口（类似 Vim 风格） |

### 窗口管理操作

| 快捷键组合               | 功能描述           |
| ------------------- | -------------- |
| `Super + m`         | 最大化当前窗口        |
| `Super + g`         | 切换当前窗口的浮动/平铺状态 |
| `Super + s`         | 切换堆叠布局         |
| `Super + y`         | 切换平铺布局         |
| `Super + o`         | 切换窗口的方向        |
| `Super + x`         | 交换窗口           |
| `Super + r`         | 窗口放大           |
| `Super + Shift + r` | 窗口缩小           |

# 切换

### 切换应用窗口

| 快捷键组合                 | 功能描述     |
| --------------------- | -------- |
| `Alt + Tab`           | 切换到下一个窗口 |
| `Alt + Shift + Tab`   | 切换到上一个窗口 |
| `Super + Tab`         | 切换到下一个窗口 |
| `Super + Shift + Tab` | 切换到上一个窗口 |


### 切换工作区窗口

| 快捷键组合                               | 功能描述             |
| ----------------------------------- | ---------------- |
| `Super + Ctrl + h/j/k/l`            | 切换到工作区的上/下/左/右窗口 |
| `Super + Ctrl + Left/Right/Up/Down` | 切换到工作区的上/下/左/右窗口 |

---

### 切换工作区

| 快捷键组合                               | 功能描述                   |
| ----------------------------------- | ---------------------- |
| `Super + 1-9`                       | 切换到工作区 1-9             |
| `Super + 0`                         | 切换到最后一个工作区             |
| `Super + Ctrl + Left/Right/Up/Down` | 切换到前/后一个工作区            |
| `Super + Ctrl + h/j/k/l`            | 切换到前/后一个工作区（类似 Vim 风格） |

### 移动窗口到工作区

| 快捷键组合                 | 功能描述           |
| --------------------- | -------------- |
| `Super + Shift + 1-9` | 将当前窗口移到工作区 1-9 |
| `Super + Shift + 0`   | 将当前窗口移到最后一个工作区 |
